
# Month 1 Coding Plan

## Week 1 - Code Kickoff

### Day 1: Title Case a Sentence
**Problem**: Write a JavaScript function that takes a sentence and returns the sentence in title case (first letter of each word capitalized).
- **Example**: 
  ```javascript
  titleCase("i love programming");  // Output: "I Love Programming"
  titleCase("javaScript is fun");  // Output: "JavaScript Is Fun"
  ```
- **Function Signature**:
  ```javascript
  function titleCase(str) {
      // Your code here
  }
  ```
